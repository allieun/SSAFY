# 🎬 Movie Archive (TMDB 기반 영화 리뷰 아카이브)

> TMDB API를 활용하여 영화 정보를 불러오고,
> 내가 본 영화에 대한 리뷰와 기록을 남길 수 있는 개인 아카이브 웹 서비스

---

## 📌 프로젝트 소개

이 프로젝트는 단순한 영화 CRUD 서비스를 넘어,
외부 영화 데이터베이스인 TMDB API를 활용하여
**영화 탐색 + 개인 리뷰 기록**을 결합한 웹 애플리케이션입니다.

왓챠피디아와 같은 서비스에서 영감을 받아,
사용자가 본 영화를 기록하고 평가할 수 있는
**개인화된 영화 아카이브**를 만드는 것을 목표로 합니다.

---

## 🎯 프로젝트 목표

* Django 기반 웹 서비스 구조 복습
* TMDB API 연동을 통한 외부 데이터 활용 경험
* 영화 데이터 + 사용자 리뷰를 결합한 서비스 설계
* 실제 서비스 형태에 가까운 기능 구현 경험

---

## 🛠️ 기술 스택

* **Backend**: Django
* **Frontend**: HTML, CSS (Bootstrap)
* **Database**: SQLite3
* **API**: TMDB (The Movie Database)
* **기타**: requests, dotenv

---

## ⚙️ 주요 기능

### 1. 영화 검색 (TMDB API)

* 영화 제목 기반 검색
* TMDB API를 통해 영화 정보 실시간 조회
* 포스터, 개봉일, 줄거리 등 표시

### 2. 영화 상세 정보

* 선택한 영화의 상세 정보 확인
* TMDB 데이터 기반 UI 구성

### 3. 영화 저장 (DB 연동)

* 검색한 영화를 내 DB에 저장
* 중복 저장 방지 (TMDB ID 기준)

### 4. 리뷰 작성

* 별점 및 리뷰 작성 기능
* 내가 본 영화에 대한 기록 관리

### 5. 리뷰 관리

* 리뷰 수정 및 삭제
* 영화별 리뷰 목록 확인

---

## 🧩 데이터 구조 (예시)

### Movie

* title
* overview
* release_date
* poster_path
* tmdb_id (unique)

### Review

* movie (FK)
* rating
* content
* created_at
* updated_at
